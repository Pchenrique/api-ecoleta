# API-Ecoleta
**Api for the ecoleta system developed using the node with express**

## requirements.
1. TypeScript.
2. NodeJS.
3. Express.
4. Celebrate.
5. knex.

## usage.

Dependencias Instaling.
```
cd api
npm install 
```

Database.
- migrations
```
npm run knex:migrate
```
- seed
```
npm run knex:seed
```
Start Eerver
```
npm run dev
```
